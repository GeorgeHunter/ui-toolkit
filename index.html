UI-Toolkit

Useful CSS

UI-Toolkit is a small collection of useful CSS that acts as a somewhat opinionated basis for common elements in a website or application.

There's no installing, no dependencies just plain Sass (which could be easily ported to even plainer CSS) that is ready to go straight into any project.

Most of the items are self explanatory, if they aren't then there's some brief usage instructions in comments at the top.

I plan to expand this as I find myself writing CSS which I think I'll probably revisit. The Toolkit pieces themselves are subject to change/ refactor.

Longer term I plan to add more information and some examples to these pages.
